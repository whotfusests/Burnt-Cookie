Burnt Cookie is a fun, lightweight cheat mod for Cookie Clicker.

üç™ Features

  • CMD-style command system with burntCookie("command")
  • Unlimited cookies, auto-buy, auto-upgrades
  • Sugar lump cheats, wrinkler/shimmer tools
  • Debug panel tools and hidden achievements
  • Bakery renaming, season switching
  • Speed control, golden cookie rain
  • Full automation toggles
  • Stylish draggable/resizable mod menu
  • Floating real-time stats HUD
  • Dark, Pastel, and Neon themes built-in
  • Resizable UI with minimizable/maximizable windows
  • Modular system (fetches features from GitHub dynamically)

üì¶ Installation

Paste the following code into your browser console while Cookie Clicker is open:

fetch('https://whotfusests.github.io/Burnt-Cookie/modules/loader.js')
  .then(res => res.text())
  .then(code => eval(code));

Or use this bookmarklet [NO CODE PASTING NEEDED, INSTANT MOD INJECTION]:

javascript:(()=>{fetch('https://whotfusests.github.io/Burnt-Cookie/modules/loader.js').then(r=>r.text()).then(t=>eval(t));})()

üíª Usage

Once installed, type:

burntCookie("help")

Example:

burntCookie("cookies max")
burntCookie("autoclicker toggle")
burntCookie("ascend auto")

Try new GUI commands like:

burntCookie("theme pastel")
burntCookie("hud toggle")
burntCookie("menu resize 400 300")
